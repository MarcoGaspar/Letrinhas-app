# Letrinhas-v3---App

<p>Requirements</p>
<ul>
	<li>NodeJS - Globally/Locally</li>
	<li>Bower - Locally</li>
</ul>

<p>Install and Run</p>
<ul>
	<li>
		<b>Install Node Modules</b>
		<br>
		npm install
	</li>
	<li>
		<b>Install Bower Modules</b>
		<br>
		bower install
	</li>
	<li>
		<b>Install Grunt</b>
		<br>
		bower install grunt
	</li>
	<li>
		<b>Install Cordova</b>
		<br>
		npm install cordova
	</li>
	<li>
		<b>Run Server</b>
		<br>
		grunt serve
	</li>
</ul>
